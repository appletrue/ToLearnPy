
1,有五个数字：1、2、3、4、5，能组成多少个互不相同且无重复数字的三位数？各是多少并打印？

```python
l = [1, 2, 3, 4, 5]
s = []
for i in l:
    for j in l:
        for k in l:
            if i != j and j != k and k != i:
                s.append(100 * i + 10 * j + k)

print(s)
print(len(s))
#layout:
[123, 124, 125, 132, 134, 135, 142, 143, 145, 152, 153, 154, 213, 214, 215, 231, 234, 235, 241, 243, 245, 251, 253, 254, 312, 314, 315, 321, 324, 325, 341, 342, 345, 351, 352, 354, 412, 413, 415, 421, 423, 425, 431, 432, 435, 451, 452, 453, 512, 513, 514, 521, 523, 524, 531, 532, 534, 541, 542, 543]
60
```
随即取样:
```python
import random

l = [1, 2, 3, 4, 5]
s = []

def samplefor():
    slice = random.sample(l, 3)
    s0 = 100 * slice[0] + 10 * slice[1] + slice[2]
    print(s0)
    return s0


samplefor()
print(s)
print(len(s))
```
用迭代器:
```python
import itertools

l = [1, 2, 3, 4, 5]
s =[i[0]*100+i[1]*10+i[2] for i in itertools.permutations(l, 3)]

print(s)
print(len(s))
```
